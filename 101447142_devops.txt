The course COMP 2156, "DevOps for System Administration," is offered at George Brown College as part of the Computer Systems Technology program. It is a 19-hour course. The course involves practical work, as evidenced by a student's assignment to create and configure an AWS Ubuntu instance, install various software, and submit screenshots of the executed commands. The program is designed with extensive industry input and offers specializations in its third year.The course COMP 2156, "DevOps for System Administration," covers the following topics:,Introduction to DevOps,Continuous Integration and Continuous Deployment (CI/CD),Infrastructure as Code (IaC),Containerization and Orchestration,Monitoring and Logging,DevOps Tools and Practices
